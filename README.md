# NexChat

A server-based chat application for Windows made using Python.

## Installation

1. Clone the Repository
```bash
git clone https://github.com/UnIntegrated-AI/NexChat.git
```

2. Run the requirements.txt file
```bash
pip install -r requirements.txt
```

## Setup
1. Set the server IP in the server and client.
```bash
HOST = "YOUR_IP" # Server IP here
```

2. Run the Server on one device.
```bash
python server.py
```
3. Run the Client on any device on the same network.
```bash
python client.py
```
